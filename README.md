# Keyboard-mouse-interaction
is your computer mouse *bricked* or battry *dies* or just *refuse* to **work**, then this program allows you to use your **mouse**, *Just by* **keyboard!**
## Keybinds
Here is the keybinds:
| Keys | Action |
|------|--------|
| Arrow keys | move a small bit (10px) |
| control + arrow keys | move a meduim size amount (50px) |
| control + alt + arrow keys | move a large amount (100px) |
